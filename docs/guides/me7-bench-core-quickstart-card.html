ME7 Core Quickstart First Power + First K-line

Use this card during bench bring-up. This is intentionally short and operational.

Scope: ECU wake-up and communication only. No sensor simulation required for this stage.

10-Minute Test Flow
Pass / Fail
  1. Power OFF. Do continuity checks and ensure no +12V-to-ground shorts.
  2. Set PSU current limit low (safe bring-up level).
  3. Apply KL30 (`T121/62`) and OBD power (`T16/16`).
  4. Enable switched feed (`T121/3`).
  5. Connect K-line tool at OBD and attempt ECU identification/read.

PASS: ECU responds over K-line and stays stable.

STOP: Unexpected current draw, hot wiring, no comms after verified wiring.
