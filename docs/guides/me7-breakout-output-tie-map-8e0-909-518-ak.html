Wiring Notes For Your Diagram
Practical Details
  1. Use a dedicated SENSOR GND rail for analog returns and tie it to main ground at one single star point.
  2. Tie ECU T121/83, MCP4728 GND, and both MCP4131 P0B returns to that sensor-ground rail.
  3. For each MCP4131 used as a variable resistor, use P0W + P0B as the active two-terminal rheostat and leave P0A unconnected.
  4. Keep each MCP4131 on shared SPI clock/data, but use separate chip-select lines so each pot can be addressed independently.
  5. Add 1k series resistors on low-level analog output lines to ECU pins (MAP/TPS/O2).
  6. Bring up one channel at a time: MAP first, then temperature channels, then TPS and O2 with live logging.
  7. Log every tie-in change in the roadmap progression log so diagram and bench wiring stay synchronized.
