Wiring Notes For Your Diagram
Practical Details
  1. Use one common ground reference between Pi, both breakouts, and ECU bench ground.
  2. For each MCP4131 used as a variable resistor, treat it as a 2-terminal rheostat: use P0W + P0B (or P0W + P0A) and leave the third terminal open.
  3. Keep each MCP4131 on shared SPI clock/data, but use separate chip-select lines so each pot can be addressed independently.
  4. Bring up one channel at a time: MAP first, then temperature channels, then unresolved channels only after worksheet promotion.
  5. Log every tie-in change in the roadmap progression log so diagram and bench wiring stay synchronized.
