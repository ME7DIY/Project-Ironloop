ME7 Bench Bring-Up Order Guide

Updated ordering and execution plan for IRONLOOP bench hardware bring-up, aligned to the latest ECU pin evidence for 8E0 909 518 AK (ME7.5 0004). Split into: order now (safe power + known-good comms path), order next (sensor signal work), and hold items (until unresolved pin behavior is bench-confirmed).

IMPORTANT: This page now follows the AK pinout confidence model. Treat `T121/1,2,3,62,43` as core. Keep `T121/21` and `T121/121` as verify-required, not mandatory buys.
Cross-check references before ordering harness parts: AK Pinout Guide, Golden Worksheet, Bench Core Quickstart.

Estimated Spend

  • Batch A (safe bring-up): roughly $600 - $1,000
  • Batch B (signal + logging): roughly $300 - $1,100
  • ME7 connector/harness parts: highly variable ($100 - $500)
  • Total practical range: about $1,000 - $2,600

Order Sequence (Recommended)

  • 1) Lock core pin worksheet (`1/2/3/62/43`, OBD `4/5/7/16`)
  • 2) Power safety parts + bench PSU
  • 3) K-line interface + continuity tools
  • 4) Pi 5 + storage + cooling
  • 5) DAC + digital pot + wire consumables
  • 6) Build harness core first, then optional/verify branches (`21/121`, CAN)

Source Map

  • Raspberry Pi compute/accessories: raspberrypi.com official reseller list
  • Analog/IC/fuse electronic parts: Digi-Key, Mouser
  • Breakout boards: Adafruit, SparkFun
  • Harness/connectors/wire: Corsa Technic and automotive wiring suppliers
  • Diagnostics: Ross-Tech official store

What Not To Buy Yet

  • Random ECU connector kits without matching your exact ME7 part number
  • Treating `T121/21` or `T121/121` as mandatory before bench verification
  • High-cost crank/cam conditioning boards before basic power + analog validation
  • Exotic instrumentation unless you already know the exact measurement gap
  • Bulk quantities of terminals until final pin map is locked
